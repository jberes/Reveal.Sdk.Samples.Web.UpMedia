The JSON files (MapData and MapStrings) should be located in the location pointed to by the MapFileDataStore class. The default location on the CDN is https://dl.infragistics.com/xplatform/reveal.

The dbf and shp files are the shape files and by default, are located based on the serverPath setting in the MapData.json file. Currently defined as http://dl.infragistics.com/xplatform/shapes/.
